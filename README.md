# Vue Shop for School Lab

本项目为GUET软件工程专业大二下学期 Web 前端课程大作业，练习之作。

---

## 一、项目所用技术栈说明

本项目采用 Vue 3 作为前端开发框架，通过 ref、onMounted 等组合式函数实现基本的响应式数据管理和生命周期控制。

项目使用 Vite 作为构建工具。

路由部分使用 Vue Router 进行页面管理，采用 History 模式，实现了登录、注册、商品列表、购物车以及订单列表等页面之间的跳转与访问。

网络请求方面使用 Axios 对 HTTP 请求进行封装。

此外，项目在 Vite 配置中设置了本地代理，将部分请求转发至后端服务器，用于解决跨域访问问题。

---

## 二、项目使用方式

### 1. 安装依赖

在项目根目录下执行：

npm install

### 2. 启动项目

依赖安装完成后，执行：

npm run dev

启动成功后，根据终端提示在浏览器中访问对应的本地地址即可查看运行效果。
